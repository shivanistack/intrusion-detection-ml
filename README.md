# intrusion-detection-ml
Overview
A Machine Learning model that detects cyberattacks 
in network traffic. Built on the NSL-KDD benchmark 
dataset using Random Forest algorithm.
 Results
 Model, Accuracy 
 Random Forest, 99.87%
